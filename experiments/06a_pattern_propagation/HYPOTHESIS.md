# Hypothesis

A single deliberate semantic smell introduced at session zero will be
reproduced by an agent in subsequent sessions without explicit reference,
at a rate demonstrating convention-latch behaviour. Additionally, agents
produce code at a volume that makes propagation effectively instantaneous
relative to human development speed.

## Prediction

- The majority of 10 subsequent agent sessions will reproduce the
  introduced naming pattern unprompted, treating it as an established
  codebase convention.
- With a controlled vocabulary in place, reproduction rate will drop
  significantly.
- Agent lines-of-code output per hour will exceed an equivalent human
  developer by 3-5x on representative tasks, confirming the volume
  acceleration multiplier described in the paper.

## Design Summary

Part 1: Introduce a deliberate semantic smell at session zero. Run 10
agent sessions on related tasks without referencing the smell. Measure
reproduction rate and file spread. Repeat with controlled vocabulary.
Part 2: Measure raw lines of code per hour, agent vs human, on matched
representative tasks.
